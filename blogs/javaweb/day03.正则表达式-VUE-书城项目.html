# 第一章.正则表达式

# 1.正则表达式概念

1.什么是正则表达式:字符串表示的匹配规则
2.作用:主要用于校验
3.比如:校验一个QQ号
       a.不能是0开头   if(startsWith("0"))
       b.只能是数字    if(字符>='0' && 字符<='9')
       c.5-15位      if(字符串.length()>=5 && 字符串.length()<=15)
    
  正则:[1-9][0-9]{4,14}   []代表区间
表达式描述
[a-z]这个字符必须是小写字母 a [] 代表区间
[abc]字符必须是abc z
[0-9]这个字符必须是数字 5
[a-zA-Z0-9]这个字符必须是字母或者是数字 [a-zA-Z0-9]
[^a-z]这个字符不是小写字母 []中写^代表取反
[\d]等同于[0-9]
[\w]等同于[a-zA-Z_0-9] 字母、数字、下划线
[\D]等同于[^0-9]
[\W]等同于[^a-zA-Z_0-9]
.代表匹配任意字符， 若只是想代表普通数据. 需要使用转义字符来表示\.
X*X这个字符可以出现零次或者多次 [0-9]* 111
X?X这个字符可以出现零次或者一次 [0-9] ?
X+X这个字符可以出现一次或者多次 [0-9] +
X{m}X这个字符出现次数正好m次 [0-9]{4}
X{m, }X这个字符出现次数至少m次 [0-9]{4, }
X{m, n}X这个字符出现次数至少m次,最多n次 [0-9]{3,5}
(X)+()括号代表X所表示的内容作为一组数据出现；()括号后面的+号，代表这组数据可以出现一次或者多次 (abc)+ -->代表的是abc可以连续出现一次或多次 (abc) + abcabc abcabcabc
^X^匹配开头
X$$匹配结尾

^如果在[]里面,代表取反

^如果在[]外面,代表以什么开头

# 2.正则表达式的创建和使用

方式1:
  var reg = /^pattern$/; //JS中,定义正则表达式规则 写在//之间
方式2:
  var reg = new RegExp("^pattern$"); //JS中,定义正则表达式规则 写在()之内

方法:
  test("字符串")--> 判断填写的字符串是否符合正则表达式

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>正则创建使用</title>
</head>
<body>
<script>
    /*
      方式1:
        var reg = /^正则表达式$/
      方式2:
        var reg = new RegExp("^正则表达式$")

      方法:
        test("要验证的字符串")

      注意:
       ^ 代表的是以xxx开头
       $ 代表的是以xxx结尾

     */
    //要求,手机号  1开头  第二位 3456789  从3-11位开始必须都是数字
    var reg1 = /^[1][3456789][0-9]{9}$/;
    var result01 = reg1.test("13838381438");
    console.log(result01);

    //需求:QQ号

    var reg2 = new RegExp("^[1-9][0-9]{4,14}$");
    var result02 = reg2.test("01111111");
    console.log(result02);
</script>
</body>
</html>

# 3.模糊匹配和精确匹配

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>正则匹配</title>
</head>
<body>
<script>
    /*
      js默认为模糊匹配
     */
    var reg1 = /abcd/;
    var result01 = reg1.test("abcde");
    console.log(result01);

    /*
      让js精准匹配
        在正则开头用^   以xxx开头
        在正则结尾用$   以xxx结尾
     */
    var reg2 = /^abcd$/;
    var result02 = reg2.test("abcd");
    console.log(result02);
</script>
</body>
</html>

# 5.正则表达式_字符集

    正则表达式-字符类:[]表示一个区间,范围可以自己定义
        语法示例：
        1. [abc]：代表a或者b，或者c字符中的一个。
        2. [^abc]：代表除a,b,c以外的任何字符。
        3. [a-z]：代表a-z的所有小写字符中的一个。
        4. [A-Z]：代表A-Z的所有大写字符中的一个。
        5. [0-9]：代表0-9之间的某一个数字字符。
        6. [a-zA-Z0-9]：代表a-z或者A-Z或者0-9之间的任意一个字符。
        7. [a-dm-p]：a 到 d 或 m 到 p之间的任意一个字符

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>字符类</title>
</head>
<body>

</body>

<script>
    //1.验证字符串是否以h开头,d结尾,中间是aeiou的某一个字符

    var regex1 = /^[h][aeiou][d]$/;
    var result01 = regex1.test("hyd");
    console.log(result01);

    //2.验证字符串是否以h开头,d结尾,中间不是aeiou中的某一个字符
    var regex2 = /^[h][^aeiou][d]$/;
    var result02 = regex2.test("had");
    console.log(result02);

    //3.验证字符串是否a-z任何一个小写字母开头,后面是ad
    var regex3 = /^[a-z][a][d]$/;
    var result03 = regex3.test("aad");
    console.log(result03);
</script>
</html>

# 7.正则表达式_预定义字符

 正则表达式-预定义字符
    语法示例：
    1. "." ： 匹配任何字符。(重点)
    2. "\d"：任何数字[0-9]的简写；(重点)
    3. "\D"：任何非数字[^0-9]的简写；
    4. "\s"： 空白字符：[ \t\n\x0B\f\r] 的简写
    5. "\S"： 非空白字符：[^\s] 的简写
    6. "\w"：单词字符：[a-zA-Z_0-9]的简写(重点)
    7. "\W"：非单词字符：[^\w]
 注意:^ 匹配字符串的开始，但在[]中使用表示取反

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>正则表达式_预定义字符</title>

</head>
<body>
<script>
    //1.验证字符串是否是三位数字
    let regex1 = /^\d\d\d$/;
    var result01 = regex1.test("111");
    document.write(result01+"<br/>");//true
    
    //2.验证手机号:1开头;第二位3  5  8  整下的9位都是0-9的数字
    let regex2 = /^[1][358]\d\d\d\d\d\d\d\d\d$/;
    let result02 = regex2.test("15010878700");
    document.write(result02+"<br/>");//true
    
    //3.验证字符串是否以h开头,以d结尾,中间是任何一个字符
    let regex3 = /^[h].[d]$/;
    var result03 = regex3.test("had");
    document.write(result03+"<br/>");//true
</script>
</body>
</html>

# 8.正则表达式_数量词

 正则表达式-数量词
        语法示例：x代表字符
        1. X? : x出现的数量为 0次或1次
        2. X* : x出现的数量为 0次到多次 任意次
        3. X+ : x出现的数量为 1次或多次 X>=1次
        4. X{n} : x出现的数量为 恰好n次 X=n次
        5. X{n,} : x出现的数量为 至少n次 X>=n次  x{3,}
        6. X{n,m}: x出现的数量为 n到m次(n和m都是包含的)   n=<X<=m

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>正则表达式_数量词</title>

</head>
<body>
<script>
    //1.验证字符串是否是3个数字
    // let regex1 = /^\d{3}$/
    let regex1 = /^[0-9]{3}$/
    var result01 = regex1.test("111");
    document.write(result01+"<br/>");
  
    //2.验证手机号:1开头;第二位3  5  8  整下的9位都是0-9的数字
    let regex3 = /^[1][358][0-9]{9}$/;
    let result03 = regex3.test("15010878111");
    document.write(result03+"<br/>");
    
    //3.验证QQ号:第一位不能为0   5-15位   全部都是数字
    let regex4 = /^[1-9][0-9]{4,14}$/;
    let result04 = regex4.test("1111111");
    document.write(result04+"<br/>");
</script>
</body>
</html>

# 9.常用正则表达式

需求正则表达式含义
用户名/^[a-zA-Z_][a-zA-Z_-0-9]{5,9}$/只能以字母或者''开头，后面的内容可以是字母、数字、''、'-'都行，整个字符串只能有6-10位
密码/^[a-zA-Z0-9_-@#&*]{6,12}$/密码只能由字母、数字、'_'、'-'、'@'、'#'、''、'&'、'*'组成，总共是6-12位
前后空格/^\s+|\s+$/g匹配开头的空格或者结尾的空格
电子邮箱/^[a-zA-Z0-9_.-]+@([a-zA-Z0-9-]+[.]{1})+[a-zA-Z]+$/多个字母、数字、'.'、'-'加上@多个数字、字母、'-'再加上'.'再加上多个字母

# 4.1.设置文本(标签体)指令_v-text和v-html

1.语法格式:  在标签中使用
  v-text = "data中的key"
  v-html = "data中的key" 
    
2.作用和区别:
  v-text相当于innerText,将vue的data中的数据放到指定的标签体中,但是不支持显示标签效果
  v-html相当于innerHTML,将vue的data中的数据放到指定的标签体中,支持显示标签效果

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>文本指令</title>
    <script src="vue.js"></script>
</head>
<body>
<div id="app">
    <div v-text="city1"></div>
    <div v-html="city2"></div>
</div>
</body>
<script>

    /*
      文本指令:
        v-text = "data中的key"
        v-html = "data中的key"

        v-text:设置标签体时,如果内容为标签,不生效
        v-html:设置标签体时,如果内容为标签,生效
     */
    new Vue({

# 8.1 VUE生命周期介绍

在我们各种语言的编程领域中，『生命周期』都是一个非常常见的概念。一个对象从创建、初始化、工作再到释放、清理和销毁，会经历很多环节的演变。比如我们在JavaSE阶段学习过线程的生命周期(线程状态)，今天学习Vue对象的生命周期，将来还要学习Servlet、Filter等Web组件的生命周期。

总结:
  1.我们只需要关注
    初始化vue对象-->用el绑定div-->挂载vue完成-->更新-->销毁
  2.最值得关注的是:挂载完成
    只要是挂载完成,指的是vue真正将div控制起来   el:"#app"->绑定div

                    this.errorMessage = "用户名不能为空";
                    //阻止表单提交
                    event.preventDefault();
                }

                //如果data中的password为空,阻止表单提交,不能登录
                if (this.password==""){
                    //给errorMessage重新赋值->密码不能为空
                    this.errorMessage = "密码不能为空";
                    //阻止表单提交
                    event.preventDefault();
                }

                //如果data中的username和password都为空,也阻止表单提交,不能登录
                if (this.username=="" && this.password==""){
                    this.errorMessage = "用户名和密码不能为空";
                    //阻止表单提交
                    event.preventDefault();
                }
            }
        }
    });
</script>

<script>
    new Vue({
        el: "#app",
        data: {
            "username":"",
            "password":"",
            "passwordConfirm":"",
            "email":"",

            "usernameMessage":"",
            "passwordMessage":"",
            "passwordConfirmMessage":"",
            "emailMessage":""
        },
        methods: {
            //校验用户名
            checkUsername(){
                //注册的用户名功能必须是3-16位的数字或者大小写字母
                var regexName = /^[0-9A-Za-z]{3,16}$/;
                //判断输入的内容是否符合用户的正则
                if (regexName.test(this.username)){
                    this.usernameMessage = "";
                    return true;
                } else{
                    //否则要有提示,证明不符合正则
                    this.usernameMessage = "用户名格式不正确";
                    return false;
                }
            },
            //校验密码
            checkPassword(){
                //注册的密码必须是6-16位的数字或者字母
                var regexPwd = /^[0-9A-Za-z]{6,16}$/;
                //判断输入的内容是否符合密码的正则
                if (regexPwd.test(this.password)){
                    this.passwordMessage = "";
                    return true;
                } else{
                    //否则要有提示,证明不符合正则
                    this.passwordMessage = "密码格式不正确";
                    return false;
                }
            },

            //确认密码
            checkPwd(){
                //判断密码框中的内容是否和确认密码框中的内容一致
                if (this.passwordConfirm==this.password){
                    this.passwordConfirmMessage = "";
                    return true;
                } else{
                    this.passwordConfirmMessage = "密码不一致";
                    return false;
                }
            },

            //校验邮箱
            checkEmail(){
                //定义邮箱的正则
                var regexEmail = /^[a-zA-Z0-9_\.-]+@([a-zA-Z0-9-]+[\.]{1})+[a-zA-Z]+$/;
                //判断输入的是否符合邮箱规则
                if (regexEmail.test(this.email)){
                    this.emailMessage = "";
                    return true;
                } else{
                    this.emailMessage = "邮箱格式不正确";
                    return false;
                }
            },

            /*
              如果以上的校验有一项格式不正确,返回false,就要阻止表单提交
             */
            checkRegister(){
                if (this.checkUsername()==false || this.checkPassword()==false || this.checkPwd()==false || this.checkEmail()==false){
                    event.preventDefault();
                }
            }
        }
    });
</script>
